# E-Commerce Product Catalog Capstone Project

## Overview

A production-ready web application demonstrating:

* Modular Frontend Architecture
* Client-Side Routing
* Responsive Design
* Asset Optimization
* Performance Best Practices
* Deployment to Vercel

## Features

### Modular Architecture

* data.js
* products.js
* router.js
* app.js

### Client-Side Routing

Routes:

* Home
* Products
* About
* 404 Page

### Performance Optimization

* Compressed Images

* Lazy Loading Images
* Responsive Assets
